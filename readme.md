Food News
=========

[Food News](http://food.hypertexthero.com) is a Hacker News clone, built using [Drum](http://drum.jupo.org/) & [Mezzanine](http://http://mezzanine.jupo.org/). Uses [Mozilla Persona](https://persona.org) for authentication.
